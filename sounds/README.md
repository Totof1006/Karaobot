# Dossier sounds/

Placer ici le fichier audio des applaudissements :

    applause.mp3

Ce fichier est joué automatiquement après chaque passage de chanteur,
pendant que les votes sont ouverts.

## Où trouver un fichier applause.mp3

- https://freesound.org (recherche "applause")
- https://pixabay.com/sound-effects/ (recherche "applause")
- N'importe quel fichier MP3 d'applaudissements de 5-15 secondes convient.

## Note Railway

Si tu déploies sur Railway, ajoute ce fichier sur GitHub AVANT de déployer.
Railway ne permet pas d'uploader des fichiers binaires directement.

Ajoute dans .gitignore une exception :
    !sounds/applause.mp3
